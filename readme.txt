INSTALLATION
--------------------

1:   Install the module
1.a:  Add the following line to the modules file:

  <module dir="custom/byggsamordning" name="byggsamordning" permissionlevel="public"></module>

2:   Edit Profile(s)

2.a: Add tools:
<tool displayname="Byggsamordning" module="byggsamordning" name="byggsamordning" panel="top-right"/>
